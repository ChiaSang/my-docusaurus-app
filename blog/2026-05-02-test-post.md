# Docusaurus 功能测试指南

这篇文章旨在展示 Docusaurus 对 Markdown 的增强支持。如果你能看到这段文字，说明你的基础配置已经生效了！

<!--truncate-->

## 1. 提示框 (Admonitions)

Docusaurus 提供了一套精美的提示框语法，用于吸引读者注意：

:::tip 提示
这是一个提示框，非常适合放置小技巧。
:::

:::danger 警告
这是一个危险警告框，通常用于提醒容易出错的操作。
:::

:::info 相关信息
这是一条普通的信息提示。
:::

## 2. 交互式代码块

你可以展示带有行高亮的代码示例：

```jsx {1,4-6}
function HighlightDemo() {
  const [count, setCount] = useState(0);
  return (
    <button onClick={() => setCount(count + 1)}>
      你点击了 {count} 次
    </button>
  );
}
```

## 3. 资源引用

在 Docusaurus 中，你可以轻松引用静态资源：

- **本地图片**: `![描述](./img/my-image.png)`
- **文档链接**: `[查看文档](../docs/intro.md)`
